
# React Chat

A real-time chat application built with modern web technologies.
## Features

- Real-time messaging
- User authentication
- Private and group chats
- Responsive design for both desktop and mobile devices


## Tech Stack

**Client:** 
- HTML,
- CSS,
- JavaScript, 
- React.js

**Server:** 
- Firebase
- JavaScript

## Deployment

To deploy this project run

- cd chat

- npm install

- npm run dev 

- open http://localhost:3000
